## Описание
Проект является учебным. В нем реализовано автоматизированное тестирование API учебного веб-сайта «Яндекс Самокат». Покрыты следующие сценарии:
* тестирование POST-запроса на создание курьера (позитивные и негативные проверки);
* тестирование POST-запроса на авторизацию курьера (позитивные и негативные проверки);
* тестирование POST-запроса на создание заказа (позитивные проверки);
* тестирование GET-запросов на получение списка заказов (позитивные проверки);
* тестирование DELETE-запроса на удаление курьера (позитивные и негативные проверки).

**Цель проекта:**
* изучение и закрепление основ автоматизированного API тестирования на языке Java с использованием RestAssured;
* демонстрация навыков использования фреймворка Allure.

**Особенности проекта:**
* используется паттерн Page Object Model;
* используется JavaFaker для генерации динамических тестовых данных;
* используется Lombok для получения более чистого и читаемого кода;
* выполняется логирование запросов и ответов;
* интеграция с Allure для формирования отчетов.

src

        ├── data
        │   ├── TestCourierData.java        #тестовые данные для работы с курьером

## Развитие
В дальнейшем планируется разработать автотесты для следующих запросов:
* GET-запрос на получение кол-ва заказов курьера (позитивные и негативные проверки);
* PUT-запрос на принятие заказа (позитивные и негативные проверки);
* GET-запрос на получение заказа по его номеру (позитивные и негативные проверки);
* GET-запрос на получение списка заказов (негативная проверка);
* PUT-запрос на отмену заказа (позитивные и негативные проверки);
* PUT-запрос на завершение заказа (позитивные и негативные проверки);
